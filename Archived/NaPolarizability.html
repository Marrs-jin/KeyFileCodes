All polarizabilities are in given atomic units; the wavelengths are in nm.

  • Click on "Static polarizability" to see a table of static polarizabilities. Click on the "Back" button to return to the main polarizability page.
  • Click on a state button (for example, 3s) to see a dynamic polarizability plot for this state.
  • Click on a button with two states (for example, 3s & 3p) to see dynamic polarizability plots for these states.
  • Use either polarizability and wavelength sliders or plot tool 'box_zoom' to rescale the graph. Click on the "reset" button to go back to the initial graph.
  • Use tool 'save' to save the graph in png format.
  • Click on "Download data" to download the data file used to plot the graph.

Plot Tools

Tool name: 'box_zoom'

The box zoom tool allows the user to define a rectangular region to zoom the plot bounds to, by left-dragging a mouse or dragging a finger across the plot area.

Tool name: 'hover'

The hover tool pops up a tooltip whenever the cursor is over a curve.

Tool name: 'crosshair'

The crosshair tool draws a crosshair annotation over the plot, centered on the current mouse position.

Tool name: 'save'

The save tool allows the user to save a PNG image of the plot.

Tools with a blue line beside them are active. By clicking on a tool you can make it active or inactive.

A polarizability curve can be shown/hidden for a state by clicking on the corresponding legend.
